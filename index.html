🟨 Kaju Board

🛠 Tools

  • V Select
  • R Rectangle
  • T Text
  • S Sketch
  • E Eraser

🧭 Navigation

  • Ctrl + Arrow Move
  • Scroll → Zoom
  • Space + Drag → Pan
  • Z / Alt → Hold for magnifier

⚡ Actions

  • Ctrl + D Duplicate
  • Ctrl + M Lock / Unlock
  • Delete Remove
  • Ctrl + Z / Y Undo / Redo

🎯 Precision

  • Shift + Ctrl + Arrow Faster
  • [ / ] Adjust eraser size
  • Ctrl + Sketch → Erase
  • Shift + Click → Add/Remove selection

💾 Files

  • Save → .kj file
  • Load → Restore board
  • Export → PNG

🎨 Extras

  • Ctrl + Click → Change color
  • Drag & Drop → Image
  • 🖼 → Image URL
💡 Pro tip: Duplicate + Lock = instant layout system
