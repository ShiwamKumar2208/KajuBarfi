Kaju Board Guide

Everything you need to use this board efficiently.

🛠 Tools

  • V → Select / Move
  • R → Rectangle
  • T → Text
  • S → Sketch (free draw)
  • E → Eraser

🖱 Interaction

  • Drag → Move element
  • Drag corners → Resize
  • Click → Select
  • Delete / Backspace → Remove

⚡ Shortcuts

  • Ctrl + D → Duplicate
  • Ctrl + M → Lock Movement / Unlock Movement
  • Ctrl + Arrow → Move precisely
  • Shift + Ctrl + Arrow → Faster move
  • Ctrl + Z / Y → Undo / Redo
  • Esc → Back to Select tool

🧭 Navigation

  • Scroll → Zoom
  • Space + Drag → Pan

🎨 Extras

  • Ctrl + Click (Rectangle) → Change color
  • Drag & Drop → Add image
  • 🖼 Button → Add image via URL
  • 🔒 Locked elements cannot be moved or resized

💾 Files

  • Save → Downloads a .kj file
  • Load → Restore previous board
  • Export → Save as PNG

💡 Tips

  • Use duplicate (Ctrl + D) instead of redrawing
  • Lock background elements to avoid accidental movement
  • Use zoom + pan for large boards
